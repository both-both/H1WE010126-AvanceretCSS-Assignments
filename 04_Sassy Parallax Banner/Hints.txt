1. Referer $parallax-height som en property for #landscape og #monkeys. Reference center-margin inden i p.

2. Leg lidt med property værdien for at forstå hvordan hver enkel af dem virker i forhold til parallax effekten. Leg især med transform property.

Kode:
#landscape, #monkeys{
  width: 100%;
  height: 100vh;
  perspective: 1px;
  overflow-x: hidden;
  overflow-y: auto;
  transform: translateZ(-1px);
}

3. Kode:
@mixin background-size($size) {
  -webkit-background-size: $size;
  -moz-background-size: $size;
  -o-background-size: $size;
  background-size: $size;
}

5. Kode:
@include parallax-background("monkeys.jpg")

6. Kode:
h1{
    transform: translateZ(-2px)
}
